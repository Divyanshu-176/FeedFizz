# Feed Fizz

Feed Fizz is a community-driven product feedback platform that helps teams collect and prioritize feature ideas in one place. Instead of feedback getting scattered across chats and messages, users can submit ideas, upvote requests they care about, and follow progress through a public roadmap.

Live demo: [https://feed-fizz.vercel.app/](https://feed-fizz.vercel.app/)

## What This Project Does

Feed Fizz creates a simple loop between users and builders:

- Users share product ideas and pain points
- The community votes so demand is visible
- The team reviews top requests and moves them into roadmap stages
- Everyone can track what is planned, in progress, and shipped

The goal is to make product decisions more transparent and data-driven by showing what users want most.

## Tech Stack

- Next.js
- React
- TypeScript
- Prisma
- Clerk Auth
- Tailwind CSS

## Why Feed Fizz

Many products receive useful feedback, but it often lives in disconnected places. Feed Fizz gives that feedback a structured workflow so teams can:

- Prioritize features based on real user demand
- Communicate roadmap direction clearly
- Build trust by showing visible progress on requests
